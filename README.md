# Faiss-FAQ_roboat
Application of Faiss in FAQ-based Question Answering
faiss 在基于FAQ检索式智能客服上的应用

### 项目介绍
  这是基于 FAQ 检索式智能客服自动问答项目中 在QA库中检索时用到的faiss检索的部分。
  此小项目 demo 仅供实验参考。

### 运行环境
  * python3.7
  * sentence-transformers==2.1.0
  * faiss==1.7.1
  其中 faiss 为 cpu 版。
  
### 效果
  10k+ 条标准question 中检索，余弦相似度为衡量标准：（其中时间单位为 s|秒）
